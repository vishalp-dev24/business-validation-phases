# Product-Market Fit Measurement - Phase 4: PMF Validation

## Sean Ellis 40% Test

### Survey Design

**The Question:**
> "How would you feel if you could no longer use [product name]?"

**Answer Options:**
- Very disappointed
- Somewhat disappointed
- Not disappointed (it really isn't that useful)
- N/A - I no longer use [product]

**Follow-up Questions:**

**If "Very Disappointed":**
> "What would you miss most if you could no longer use [product]?"
[Open text]

**If "Somewhat" or "Not Disappointed":**
> "What could we do to make [product] more useful for you?"
[Open text]

**For all users:**
> "Have you recommended [product] to anyone?"
- Yes, I've actively told others about it
- Mentioned it when asked
- No

> "How would you describe [product] to a friend?"
[Open text - analyze for positioning clarity]

> "What type of person would benefit most from [product]?"
[Open text - analyze for persona clarity]

## Survey Administration

### Minimum Sample Size
- **Target: 40+ respondents**
- Minimum: 30 respondents
- Include only users who:
  - Have experienced core value proposition
  - Used product at least twice
  - Used within last 2 weeks

### Distribution Strategy
- In-app survey (highest response rate)
- Email to active users
- Exclude brand new users (<1 week)
- Exclude churned users (>30 days inactive)

### Timing
- Run survey every 6-8 weeks
- Track trend over time
- Compare cohorts (new vs. old users)

## First Round PMF Method

### PMF Level Assessment

| Level | Indicators | Your Status |
|-------|------------|-------------|
| **Nascent** | First paying customers, usage pattern emerging | |
| **Developing** | Repeatable sales, early referrals | |
| **Strong** | Inbound > outbound, "fish jumping in boat" | |
| **Extreme** | Dominant position, category-defining | |

### Anti-Pattern Detection

**Signals of Real PMF (not vanity):**
- [ ] Customer complaints (they care enough to complain)
- [ ] Competitors copying features
- [ ] Organic mentions without asking
- [ ] 40%+ direct traffic
- [ ] Decreasing CAC over time

**Signals of False PMF:**
- [ ] Growth only from paid acquisition
- [ ] High churn masked by new user growth
- [ ] Users acquired but not activated
- [ ] Feature requests for basic functionality

## The 4 P's of PMF (First Round)

### Analysis Template

**PERSONA:** Who specifically?
- [ ] Primary persona clearly defined
- [ ] Secondary personas identified
- [ ] Persona characteristics validated with data

**PROBLEM:** What urgent pain?
- [ ] Problem ranked top 3 for persona
- [ ] Problem occurs frequently (>weekly)
- [ ] Current workarounds are inadequate

**PROMISE:** What outcome?
- [ ] Promise is specific and measurable
- [ ] Promise delivered within first use
- [ ] Promise aligns with customer language

**PRODUCT:** What solution?
- [ ] Solves the specific problem
- [ ] Personas can self-serve
- [ ] Delivers on promise consistently

## Organic Growth Analysis

### Word of Mouth Coefficient

**Formula:**
```
Viral Coefficient = (New users from referrals) / (Total active users)
```

**Benchmarks:**
- <0.5: Mostly paid growth
- 0.5-1.0: Some organic, mostly paid
- >1.0: True virality (exponential potential)
